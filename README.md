provide estimated class labels (0 or 1) for every trial of the test data

## Environment & package required

### Environment

1. Windows10 x64
2. Python 3.5.6
3. Nvidia CUDNN v9.0 if tensorflow-gpu has been chosen

### packages

1. tensorflow or tensorflow-gpu V1.11 or above
2. numpy
3. pandas
